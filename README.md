# ¡Hola Español! | Spanish teacher Website

[English](#english) | [Polski](#polski)

---

## English

Professional portfolio website for Ania Kowalska, a Spanish teacher. The project focuses on clear communication of individual and small group lesson offerings (Warsaw & Online).

### 🚀 Key Features
- **Responsiveness**: Fully mobile-friendly via Bootstrap 5.3.
- **Branding**: Custom color palette (Navy & Turquoise) and personal signature (Reenie Beanie font).
- **Interactivity**: Auto-switching student testimonials carousel.
- **Optimization**: Clean code, modern typography, and Lucide icons.

### 🛠 Tech Stack
- HTML5 / CSS3
- [Bootstrap 5.3](https://getbootstrap.com/)
- [Lucide Icons](https://lucide.dev/)
- [Google Fonts](https://fonts.google.com/)

### 💻 How to run locally
1. Clone the repository.
2. Open `index.html` or start a server: `python3 -m http.server 8000`.

> **Note**: This website was created in the **Wibecoding** style using the **Google Antigravity** tool.
